Use the Edit Links for ACF Relationship Field plugin to add edit links to the administration of ACF Relationship Fields.

== Description ==
This plugin adds edit links to the posts in the administration of the ACF Relationship Field.

= Requirements =

* Advanced Custom Fields 5+ (ACF 4 is not supported)
* PHP 5.6+

== Installation ==
From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Edit Links for ACF Relationship Fields"
3. **Activate** Edit Links for ACF Relationship Fields from your Plugins page
4. **Create** a Relationship Field with ACF
5. **Enjoy** the new post edit links
